## 1. 系统定位

本系统是一个 **虚拟电厂（VPP）聚合商视角的光伏运营管理平台**。它不是单个电站的监控页面，而是把多个光伏电站组织为"资源单元"进行统一管理，覆盖从实时监控、功率预测、调度策略到收益分析的完整业务闭环。

系统面向的核心角色：

| 角色 | 关注点 |
|---|---|
| 运营调度人员 | 实时出力、调度响应、告警处理 |
| 预测分析师 | 功率预测精度、偏差分布、可调能力 |
| 策略管理人员 | 策略配置、收益模拟、电价窗口优化 |
| 管理层 | 经营 KPI、发电排名、收益汇总 |

## 3. M01 综合监控中心

### 3.1 功能概述

综合监控中心是系统首页，以"驾驶舱"形态呈现，一屏聚合展示所有电站的全局运营状况。管理层和调度人员打开系统后首先看到的就是这个页面。

### 3.2 页面区域

#### 3.2.1 GIS 电站地图

- 在高德地图上以散点形式展示所有电站的地理位置
- 每个散点的颜色对应电站状态：绿色（正常）、黄色（告警）、红色（故障）、蓝色（检修）、灰色（离线）
- 支持三个维度的筛选：**状态**、**区域**、**容量范围**
- 点击地图上的散点可聚焦该电站，联动刷新 KPI、曲线和告警区域
- 地图右侧显示区域汇总摘要（总功率、总容量、健康率等）
- 地图下方展示 VPP 节点可调容量状态

#### 3.2.2 经营指标看板（KPI Board）

展示所选电站（或全系统）的核心运营数字：

| 指标 | 含义 |
|---|---|
| 实时功率 (kW) | 当前瞬时输出功率 |
| 今日发电量 (kWh) | 当天累计发电量 |
| 今日收益 (元) | 当天累计上网收入 |
| 等效利用小时 (h) | 今日发电量 ÷ 装机容量 |
| 系统可用率 (%) | 可运行时间占比 |
| 可上调容量 (kW) | VPP 可向上调节的功率空间 |
| 可下调容量 (kW) | VPP 可向下调节的功率空间 |

#### 3.2.3 功率曲线

- 折线图展示某电站当天的 15 分钟粒度功率曲线
- 包含实际出力、预测出力、基线出力三条线
- 叠加展示可调上下界的面积带
- 支持日期切换，查看历史功率

#### 3.2.4 发电排名

- 以柱状图展示所有电站在某一指标维度上的排名
- 支持切换排名指标：发电量、功率、收益、可用率等

#### 3.2.5 实时告警滚动

- 展示最新告警事件列表，按告警级别（严重/重要/一般/提示）着色
- 支持按级别过滤
- 点击某条告警可展开详情抽屉，查看告警描述、建议处理方案和责任人

## 4. M02 生产监控

### 4.1 功能概述

生产监控以"资源单元"为核心维度。一个资源单元是若干地理位置相近的电站的管理集合，用于统一调度响应。该模块提供三个子视图：**出力概况**、**负荷与出力**、**调度执行**。

### 4.2 资源单元选择

- 页面顶部的筛选栏支持按区域、城市、资源单元名称进行联动筛选
- 选中资源单元后，所有子视图数据联动刷新
- 页面展示当前资源单元的状态摘要（名称、区域、状态、调度模式、策略标签）

### 4.3 出力概况（Overview）

| 展示内容 | 说明 |
|---|---|
| 资源单元健康状态 | 在线率、告警数量、最新告警标题 |
| 成员电站汇总表 | 列出该资源单元下所有电站的容量、权重、状态、在线率、告警数 |
| 功率曲线 | 15 分钟粒度的 PV 出力、预测出力和基线曲线 |
| 天气快报 | 当前天气、温度、辐照度、湿度、风速和气象结论 |

### 4.4 负荷与出力（Load）

- 以双轴折线图展示负荷曲线和光伏出力曲线的对比
- 叠加辐照度和温度参考线
- 可选择日期和数据粒度（15 分钟 / 1 小时）
- 直观展示光伏自消纳能力和可调空间

### 4.5 调度执行（Dispatch）

| 展示内容 | 说明 |
|---|---|
| 调度指令列表 | 当日所有调度指令：指令类型、目标功率、实际功率、响应时间 |
| 执行成功率 | 成功 / 偏差 / 失败的比例统计 |
| 偏差原因分析 | 执行偏差的原因分类说明 |
| 趋势图 | 调度指令的历史执行趋势 |

## 5. M02b 电站监控

### 5.1 功能概述

电站监控是生产监控的第二个子模式，以树状结构下钻到**公司 → 电站 → 逆变器**三个层级，查看每个节点的详细实时数据。

### 5.2 树状导航

左侧树面板展示企业层级结构：
```
├── 深圳科技集团
│   ├── 深圳湾科技园光伏电站
│   │   ├── INV-SZ-001 逆变器
│   │   └── INV-SZ-002 逆变器
│   └── 前海商务区光伏电站
│       └── ...
├── 杭州绿能集团
│   └── ...
```

支持按状态过滤树节点（仅显示告警/故障设备）。

### 5.3 公司层级

选中公司节点时，右侧面板展示：
- 公司下所有电站列表（容量、状态、数据质量等级）
- 可点击跳转到具体电站

### 5.4 电站层级

选中电站节点时，右侧面板展示多个 Tab 视图：

| Tab | 内容 |
|---|---|
| 可调空间 | 实时可调上下界曲线 |
| 光伏出力 | 实际 vs 预测的 15 分钟出力曲线 |
| 负荷监控 | 负荷曲线及基线对比 |
| 出力预测 | 日前预测和超短期预测曲线 |
| 执行策略 | 当前电站正在执行的策略（名称、类型、目标功率、预计收益） |

支持切换日期、数据粒度（1 分钟 / 15 分钟 / 1 小时），以及自动刷新间隔。

### 5.5 逆变器层级

选中逆变器节点时，展示设备台账信息：

- 设备基本信息：型号、制造商、序列号、固件版本、安装日期
- 电气参数：额定功率、直流输入电压、交流输出电压、电网频率
- 温度信息：模块温度、环境温度
- 组件配置：MPPT 通道数、组串数、每串组件数
- 告警记录：该逆变器的历史告警列表

## 6. M03 预测分析

### 6.1 功能概述

预测分析模块面向预测分析师和调度人员，提供功率预测结果的可视化展示、精度评估和可调能力分析。该模块提供三个子视图：**预测总览**、**可调能力分析**、**精度评估**。

### 6.2 预测总览（Overview）

| 展示内容 | 说明 |
|---|---|
| 预测对比曲线 | 日前预测 vs 超短期预测 vs 实际出力的三线对比 |
| 偏差热力图 | 以时间×日期矩阵展示预测偏差的空间分布 |
| 电站预测明细 | 各电站的预测精度指标一览表 |

筛选条件支持按区域、电站和预测类型（日前 / 超短期）过滤。

### 6.3 可调能力分析（Adjustable）

- 24 小时可调能力预测曲线：展示未来一天每个时段的可上调/可下调空间
- VPP 节点可调时间线：节点维度的可调容量区间
- 电站可调明细表：各站点的可调空间分解

### 6.4 精度评估（Accuracy）

| 指标 | 含义 |
|---|---|
| MAE (kW) | 平均绝对误差 |
| RMSE (kW) | 均方根误差 |
| 准确率 (%) | 预测准确率 |

- 月度精度趋势图：展示历史各月的 MAE/RMSE/准确率变化
- 偏差分布直方图：展示误差的分布形态
- 电站精度排名：按准确率从高到低排列各电站

## 7. M04 策略管理

### 7.1 功能概述

策略管理模块面向策略制定人员，涵盖从策略创建、审批下发到收益回测的全流程。提供三个子视图：**策略列表**、**策略配置**、**收益分析**。

### 7.2 策略列表（List）

#### 7.2.1 KPI 指标卡

页面顶部展示策略维度的全局指标：
- 策略总数、执行中数量、待审批数量
- 总预估收益、总实际收益、总偏差率

#### 7.2.2 策略台账表

| 字段 | 说明 |
|---|---|
| 策略名称 | 策略的业务标识 |
| 所属电站 | 绑定的电站 |
| 类型 | 削峰填谷 / 需求响应 / 调频辅助 |
| 状态 | 草稿 → 待审批 → 执行中 → 已完成 / 已终止 |
| 模式 | 自动执行 / 半自动 / 手动确认 |
| 目标功率 (kW) | 策略目标输出功率 |
| 起止时间 | 策略有效时间窗口 |
| 版本号 | 修改迭代版本 |

支持的操作：
- **创建策略**：填写电站、类型、时间、目标功率等参数
- **批量创建**：一次为多个电站创建同类策略
- **提交审批**：将草稿策略提交到审批流
- **终止策略**：强制终止执行中的策略
- **批量提交 / 批量删除**

### 7.3 策略配置（Config）

#### 7.3.1 策略树

以公司 → 电站 → 策略的层级展示所有策略的归属关系。

#### 7.3.2 分时电价

以柱状图展示所选电站的分时电价窗口：
- 尖峰时段（红色）
- 峰时段（橙色）
- 平时段（蓝色）
- 谷时段（绿色）

每个时段标注单价（元/kWh）。

#### 7.3.3 策略模拟

- 输入策略参数后，调用后端模拟引擎计算预估收益
- 模拟结果包含：预估日收益、置信区间（低/高）、成功概率
- 支持批量模拟：同时对多个策略进行模拟对比

### 7.4 收益分析（Revenue）

| 展示内容 | 说明 |
|---|---|
| 收益汇总 | 区域/类型维度的总预估收益、总实际收益、偏差分析 |
| 收益明细 | 以日为粒度展示收益拆分：削峰节省、响应奖励、偏差罚款 |
| 策略对比 | 选择多个策略进行收益横向对比（柱状图或表格） |

## 8. M05 设备监控

### 8.1 功能概述

设备监控提供全系统设备的状态总览和运维辅助。

### 8.2 页面区域

#### 8.2.1 指标卡

- 设备总数、在线数量、离线数量、告警设备数

#### 8.2.2 设备分布

- 以条形图展示各设备类型的占比（逆变器、汇流箱、气象站、通讯终端等）

#### 8.2.3 运维提示

- 系统根据当前设备状态生成的值班建议列表
- 例如："3 台逆变器温度超过 60°C，建议安排巡检"

#### 8.2.4 设备列表

| 字段 | 说明 |
|---|---|
| 设备编号 | 唯一标识 |
| 设备名称 | 设备业务名称 |
| 所属站点 | 归属电站 |
| 类型 | 设备类别 |
| 状态 | 在线 / 离线 / 告警 |
| 负荷率 | 当前负载占额定容量的比例 |
| 温度 | 设备温度读数 |
| 最后上报时间 | 最近一次数据上报的时间戳 |

支持按关键字和状态筛选。

## 9. M06 告警中心

### 9.1 功能概述

告警中心是所有告警事件的统一入口，支持严重、重要、一般和提示四个级别的告警管理。

### 9.2 页面区域

#### 9.2.1 指标卡

- 今日告警总数、严重告警数、待处理数、已关闭数

#### 9.2.2 处理看板

- 展示当前告警处置规则列表
- 如："严重告警 15 分钟内必须响应"、"同站点 3 次同类告警自动升级"

#### 9.2.3 告警列表

| 字段 | 说明 |
|---|---|
| 告警编号 | 唯一标识 |
| 站点 | 发生告警的电站 |
| 设备 | 发生告警的设备 |
| 分类 | 通讯中断 / 功率异常 / 温度超限 / 电网波动 等 |
| 级别 | 严重 / 重要 / 一般 / 提示 |
| 状态 | 待派单 → 处理中 → 待确认 → 已关闭 |
| 责任人 | 告警处理责任人 |
| 发生时间 | 告警事件的时间戳 |

筛选条件：关键字、级别、状态。支持分页。

## 10. 名词术语表

| 术语 | 英文 | 解释 |
|---|---|---|
| 虚拟电厂 | VPP (Virtual Power Plant) | 通过聚合分布式能源资源，以软件调度代替物理电厂的模式 |
| 资源单元 | Resource Unit | 地理上相近的若干电站组成的管理集合，作为调度响应的最小单位 |
| 日前预测 | Day-Ahead Forecast | 提前 24 小时对次日功率的预测 |
| 超短期预测 | Ultra-Short-Term Forecast | 提前 15 分钟至 4 小时的功率预测 |
| 削峰填谷 | Peak Shaving & Valley Filling | 在电价高峰时减少用电、低谷时增加用电的策略 |
| 需求响应 | Demand Response | 根据电网调度指令调整用电负荷的行为 |
| 可调容量 | Adjustable Capacity | 系统在当前状态下能够上调或下调的功率空间 |
| MAE | Mean Absolute Error | 预测值与实际值绝对差的平均值 |
| RMSE | Root Mean Square Error | 预测误差的均方根，对大偏差更敏感 |
| 装机容量 | Installed Capacity (kWp) | 光伏系统在标准条件下的最大输出功率 |
| 等效利用小时 | Equivalent Hours | 发电量 ÷ 装机容量，衡量发电效率的标准化指标 |
| PR | Performance Ratio | 实际发电量与理论发电量的比值 |
