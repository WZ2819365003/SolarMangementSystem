## 目录说明

- `handover/`
  面向交接和日常维护的主文档

- `audits/`
  审计结论、风险记录、联调问题

- `modules/frontend/`
  前端模块结构、页面职责、接口依赖

- `modules/backend/`
  后端接口、数据表、计算边界、维护建议

- `product/`
  设计资料和交付资料

- `testing/`
  测试策略、记录和验证文档

- `superpowers/specs/`
  设计 spec

- `superpowers/plans/`
  实施计划
