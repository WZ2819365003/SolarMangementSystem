## 6. 常见问题

### 问题 1：前端启动了，但页面没数据

先查：

- `frontend/src/shared/host/bridge.js`
- 当前是否误设置了 `VUE_APP_REQUEST_MODE=mock`
- `/api` 代理是不是能打到 `8091`

### 问题 2：接口有数据，但中文显示乱码

这是 PowerShell / 控制台编码现象，不一定是接口真的乱码。  
优先用浏览器、前端页面或 API 工具确认，不要只看终端输出。

### 问题 3：地图报高德资源请求

像 `jsapi-data*.amap.com` 这类请求是高德底图资源。  
这类问题要先区分：

- 是底图资源问题
- 还是你自己的后端点位数据问题

### 问题 4：H2 数据丢了

当前 H2 是内存库，重启后会重新执行 `schema.sql` 和 `data.sql`。  
所以“数据恢复到种子状态”是预期行为，不是 bug。

### 问题 5：前端路由直达 404

先确认：

- 你打开的是正确的前端实例
- Vue 开发服务是否在正确 worktree 启动
- `history` fallback 是否正常
