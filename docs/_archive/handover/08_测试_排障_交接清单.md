## 4. 现在怎么排障

### 第一步：先判断是否真的打到了后端

优先直接访问 `8091`。

如果 `8091` 正常但页面没数据，再看：

- 前端代理
- 页面筛选状态
- 字段消费

### 第二步：确认 H2 是否真的有数据

打开：

- `http://127.0.0.1:8091/h2-console`

重点查看这些表组：

- `dashboard_*`
- `pm_*`
- `sa_*`
- `fc_*`
- `sg_*`

### 第三步：区分问题发生在元数据层还是计算层

更像元数据问题的现象：

- 下拉选项缺失
- 电站或公司数量不对
- 电价表为空
- 策略列表为空

更像后端计算问题的现象：

- KPI 不合理
- 模拟收益异常
- 置信区间全 0
- 对比结果差值异常
- 树统计与列表状态不一致

## 5. M04 特别注意点

`strategy` 当前是典型的“元数据 + 后端计算”模块：

- `sg_*` 表保存开发态元数据
- `sa_* / fc_*` 提供系统事实数据
- `StrategySimulationService` 和 `StrategyRevenueService` 负责计算

所以排障时至少要同时看：

- `StrategyController`
- `StrategyDataService`
- `StrategySimulationService`
- `StrategyRevenueService`
- `Strategy*Repository`
- `schema.sql`
- `data.sql`
- 前端 `StrategyPage.vue`
