## 0. 模块定位与边界

### 0.1 与 VPP 平台的关系

本模块是虚拟电厂运营平台（HSSJ_vpp）的子应用之一，通过 qiankun 微前端架构挂载到宿主平台。**每个光伏电站作为 VPP 的一个独立可调资源节点**，需要向 VPP 平台输出可调能力、功率预测等关键数据，同时接收并执行 VPP 下发的调度策略。

**核心业务链路**：
```
光伏出力（实时+预测） → 负荷情况 → 可调空间（负荷 - 光伏出力） → 策略执行 → 收益结算
```

### 0.2 资源层级模型（V4.0 修正）

> ⚠️ **V4.0 重要变更**：一个电站 = 一个 VPP 资源节点，不再存在"资源单元聚合多站"的中间层。

```
公司（代理用户）
├── 电站 A（= 资源节点 A）
│   ├── 逆变器 INV-01
│   ├── 逆变器 INV-02
│   └── ...
├── 电站 B（= 资源节点 B）
│   └── ...
└── ...
```

**可调能力计算**：
```
可调空间 = 电站负荷 - 光伏出力
         = (用电总负荷) - (逆变器实时总出力)
```

### M02-B 电站监控（详见 M02_02）

左侧树面板 + 右侧数据面板，三级钻取：

#### 公司层级

- **9 个 KPI 卡片**：装机容量、实时总出力、总负荷、可调容量、日发电量、月发电量、在线率、告警数、VPP 节点数
- **下属电站 Tab**：表格展示所有下属电站的实时运行数据（装机、出力、负荷、可调空间、在线率等）
- **策略查看 Tab**：表格展示所有电站的策略执行情况，点击行可展开策略详情抽屉

#### 电站层级

- **7 个 KPI 卡片**：装机容量、实时出力、负荷、可调空间、日发电量、逆变器在线率、告警数
- **月度 VPP 统计条**：月度参与 VPP 调度次数、响应成功率、月累计收益
- **工具栏**：日期选择 + 时间粒度切换（5min/15min/1h）
- **5 个 Tab 页**：

| Tab | 已实现内容 |
|---|---|
| 可调空间 | 可调空间 = 负荷 - 光伏出力，面积图展示，标注上下限 |
| 光伏出力 | 实时出力曲线 + 各逆变器出力占比 |
| 负荷监控 | 负荷曲线 + 峰谷标注 |
| 出力预测 | 日前预测 vs 实际对比曲线 + 偏差区间 |
| 策略查看 | 策略列表表格（只读），点击展开策略详情抽屉 |

#### 策略详情抽屉

- **基本信息区**：策略名称、执行日期、状态、创建时间等
- **分时电价区**：峰/平/谷 三档电价展示
- **执行概要区**：计划执行量、实际执行量、偏差率、收益

#### 逆变器层级

- **KPI 卡片**：额定功率、实时出力、日发电量、运行状态、累计发电
- **拓扑图**：逆变器在电站拓扑中的位置
- **告警信息**：当前活跃告警列表
- **功率曲线**：逆变器实时功率输出曲线

## M04. 策略管理

> **定位**: 配置电站响应策略 + 接入电价信息 + 模拟收益测算
> **状态**: 📋 设计中
> **参考**: 储能管理系统「响应支援」页面的步骤式配置流程

### 页面结构

**方案：树+配置面板**（与 M02 电站监控同构）

```
┌──────────────┬──────────────────────────────────────┐
│ [左侧树面板]  │ [右侧策略配置面板]                      │
│ 公司          │                                       │
│ ├── 电站A    │  步骤 1: 响应信息录入                    │
│ │   ├ INV-01 │  ┌─────────────────────────────────┐  │
│ │   └ INV-02 │  │ 策略执行日期: 2026-03-26          │  │
│ ├── 电站B    │  │ 计划时段: 00:00 至 01:00 [+][-]   │  │
│ └── 电站C    │  │ 电价(元/kWh):                     │  │
│              │  │  00:00-00:15  00:15-00:30  ...    │  │
│              │  │  [0.35]      [0.35]       ...    │  │
│              │  │ [自动获取电价] [手动录入]            │  │
│              │  └─────────────────────────────────┘  │
│              │                                       │
│              │  步骤 2: 策略生成                       │
│              │  步骤 3: 运行效果概览                    │
│              │                                       │
│              │  [取消]                    [下一步]    │
└──────────────┴──────────────────────────────────────┘
```

### 核心功能

| # | 功能 | 说明 |
|---|---|---|
| 1 | 电站树选择 | 复用 M02 电站监控的三级树组件，选中电站后加载该站策略 |
| 2 | 步骤式策略配置 | 参考储能管理系统：响应信息录入 → 策略生成 → 运行效果概览 |
| 3 | 电价接入 | 自动获取（对接电力交易中心 API）+ 手动录入，按 15min 粒度 |
| 4 | 策略列表管理 | 表格展示已配置策略，支持弹窗编辑（备选方案） |
| 5 | 收益模拟 | 基于 M03 预测数据 + 电价 + 策略参数，计算预估收益 |
| 6 | 策略对比 | 选择 2-3 个策略并排对比预估收益 |
| 7 | 执行监控 | 实时跟踪策略执行情况、偏差、收益 |
| 8 | 收益统计 | 按场景/月度统计历史收益 |

### 电价信息接入

| 获取方式 | 说明 |
|---|---|
| 自动获取 | 对接电力交易平台 API，按日前/实时市场获取分时电价（96 点/天） |
| 手动录入 | 运营人员在表单中逐时段录入电价，支持批量填充 |
| 实时更新 | 实时市场出清价格自动更新，覆盖日前预估 |

### 与 M02 的关系

M02 电站监控的「策略查看」Tab 和策略详情抽屉为 M04 策略数据的只读展示视图。M04 负责策略的完整生命周期管理（创建/编辑/执行/统计），M02 仅展示执行结果。

## 附录 D：VPP 节点能力字段

| VPP 字段 | 中文名 | 计算方式 |
|---|---|---|
| `deferrableLoad` | 可调负荷(kW) | 当前负荷 - 光伏出力 |
| `maxUpCapacity` | 最大上调功率 | 储能可放电 + 光伏可增发 |
| `maxDownCapacity` | 最大下调功率 | 当前出力 + 储能可充电 |
| `maxUpRate` | 上升速率 | 设备参数 |
| `maxDownRate` | 下降速率 | 设备参数 |
| `ratedPower` | 额定功率 | 铭牌参数 |
| `operatingCapacity` | 运行容量 | 实时计算 |
