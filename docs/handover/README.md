# 交接手册

这组文档面向第一次接手这个项目的维护者。目标不是介绍概念，而是让接手人能快速回答三件事：

- 系统现在做到哪一步了。
- 前后端怎么启动、怎么排障、哪些链路是真实链路。
- 下一阶段要从哪里继续推进。
