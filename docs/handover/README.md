# 交接手册入口

这组文档是给“第一次接手这个系统的人”准备的。

阅读目标不是理解概念，而是尽快回答下面几件事：

1. 系统现在到了什么阶段
2. 前后端怎么启动、怎么联调、怎么排障
3. 哪些模块已经切到 H2 后端链路
4. 哪些指标是后台实时计算，哪些只是元数据
5. 接手后下一步应该从哪里继续推进
