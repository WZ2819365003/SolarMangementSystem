# 交接手册入口

这组文档面向第一次接手本项目的维护者。

阅读目标不是“了解概念”，而是尽快回答下面几件事：

1. 这个系统现在处于什么阶段
2. 前后端怎么启动、怎么联调、怎么排障
3. 哪些模块已经切到 H2 后端链路
4. 哪些模块仍然处于 mock 或半成品阶段
5. 下一步应该从哪里继续推进
